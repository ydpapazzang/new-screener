## 1. 개요

이 스크리너는 **전날 마감된 일봉(EOD)** 데이터를 기반으로 5가지 눌림목 전략 조건에 맞는 코인을 자동 탐색합니다.  
단순 신호 탐색에 그치지 않고, 각 종목에 대해 **워크 포워드 테스트 → 리스크 분석 → 복리 시뮬레이션**까지 연결된 분석 파이프라인을 제공합니다.

```
거래소 선택 → 전략 선택 → 스캔
    ↓
신호 종목 목록 확인 (승률 / 기댓값 / 순수익률)
    ↓
종목 클릭 → 차트 + 매매 플랜
    ↓
워크 포워드 테스트 (전략 강건성 검증)
    ↓
리스크 분석 (샤프/소르티노/에쿼티 커브)
    ↓
복리 시뮬레이션 (켈리 공식 + CAGR)
```

## 2. 기본 사용 흐름

### Step 1 — 거래소 & 전략 선택
페이지 상단 필터 바에서 거래소(Upbit / Binance / Bithumb)와 전략 번호(1~5)를 선택합니다.

### Step 2 — 스캔 실행
**[스캔 시작]** 버튼을 클릭합니다.  
거래소 전 종목의 1~2년치 OHLCV 데이터를 수집하고 조건을 필터링합니다.  
⚠️ 처음 실행 시 **1~3분** 소요될 수 있습니다 (이후 6시간 캐시).

### Step 3 — 종목 선택
결과 테이블에서 관심 종목 행을 클릭합니다.  
우측에 **차트 → 워크포워드 → 리스크 → 복리** 패널이 순서대로 나타납니다.

### Step 4 — 분석 실행
각 패널의 실행 버튼을 눌러 심층 분석을 수행합니다 (패널별 개별 API 호출).

## 3. 거래소 & 전략 선택

| 거래소 | 기본 통화 | 기본 수수료 | 비고 |
|--------|-----------|------------|------|
| **Upbit** | KRW | 0.05% + 슬리피지 0.10% | 국내 1위, KRW 마켓 |
| **Binance** | USD | 0.10% + 슬리피지 0.08% | 글로벌 최대 유동성 |
| **Bithumb** | KRW | 0.25% + 슬리피지 0.15% | 국내 2위 |

거래소를 바꾸면 **수수료 설정과 표시 통화가 자동으로 전환**됩니다.

## 4. 스크리닝 결과 테이블

| 컬럼 | 설명 |
|------|------|
| **종목** | 거래 페어 (예: BTC/KRW) |
| **현재가** | 전날 마감 종가 |
| **진입 대기가** | 전략이 계산한 이상적 진입 가격 |
| **백테 승률** | 최근 1년 신호 기준 승률 (%) |
| **순수익** | 수수료·슬리피지 반영 후 익절 순수익 (%) |
| **순손실** | 수수료 반영 후 손절 순손실 (%) |
| **기댓값 (EV)** | `승률 × 순수익 + 패율 × 순손실` |

**테이블 정렬**: 컬럼 헤더를 클릭하면 오름/내림차순 정렬됩니다.  
**기댓값이 양수(+)인 종목**을 우선적으로 검토하세요.

## 5. 수수료/슬리피지 설정

스크리닝 결과 테이블 위의 **[수수료/슬리피지 설정]** 패널을 펼쳐 설정합니다.

- **수수료율**: 거래소 거래 수수료 (%)
- **슬리피지**: 호가창 충격 비용 — 시장가 진입 시 실제 체결가 차이 (%)
- **왕복 비용**: `(수수료 + 슬리피지) × 2` — 매수·매도 합산 실제 비용

### 왕복 비용이 중요한 이유
백테스트 승률이 60%라도 왕복 비용 0.7%면 손익분기 승률을 초과해야 실제 수익입니다.  
패널 하단의 **EV 시뮬레이션 바**가 40~70% 구간별 기댓값을 시각화해 줍니다.

## 6. 통화 전환 (KRW ↔ USD)

테이블 우측 상단의 **[₩ 원화] / [$ 달러]** 버튼으로 전환합니다.  
환율은 Upbit KRW-USDT 실시간 시세를 사용하며, 옆의 🔄 버튼으로 수동 갱신할 수 있습니다.

- **Upbit / Bithumb**: 기본 KRW → USD 환산 시 `가격 ÷ 환율`
- **Binance**: 기본 USD → KRW 환산 시 `가격 × 환율`

## 7. EOD 차트 & 매매 플랜

종목을 클릭하면 우측에 **TradingView 스타일 캔들스틱 차트**가 표시됩니다.

### 차트 레이어
| 색상 | 의미 |
|------|------|
| 🔵 파란 점선 | 진입 대기가 |
| 🟢 초록 점선 | 1차 목표 (+10%) |
| 🟩 연초록 점선 | 2차 목표 (+20%) |
| 🔴 빨간 점선 | 손절가 (-5%) |

### 매매 플랜 카드
차트 아래 카드에서 **수치 기반 매매 플랜**을 확인합니다:
- **진입 대기가**: 해당 전략 기준 이상적 매수 시점 가격
- **1차 목표 (+10%)**: 첫 번째 익절 목표
- **2차 목표 (+20%)**: 두 번째 익절 목표  
- **손절가 (-5%)**: 리스크 관리 기준선

> 전략 시그널 상세 설명이 카드 하단에 텍스트로 표시됩니다.

## 8. 워크 포워드 테스트

**[워크 포워드 테스트 실행]** 버튼을 눌러 전략의 **실제 강건성**을 검증합니다.

### 원리
과거 데이터를 **훈련(180일) / 검증(60일)** 윈도우로 분리하고, 훈련 기간의 승률이 검증 기간에서도 재현되는지 확인합니다.  
총 5개 슬라이딩 윈도우로 최대 300일을 검사합니다.

### 효율성 지수 (Efficiency Ratio)
```
효율성 = 검증 기간 평균 승률 ÷ 훈련 기간 평균 승률
```

| 판정 | 효율성 | 의미 |
|------|--------|------|
| ✅ 강건 | ≥ 0.8 | 훈련 승률이 실전에서도 80% 이상 재현 |
| 🟡 양호 | 0.6~0.79 | 다소 낮지만 허용 범위 |
| ⚠️ 주의 | 0.4~0.59 | 실전 성과 저하 가능성 |
| ❌ 과적합 | < 0.4 | 역사적 데이터에만 맞춰진 허상 승률 |
| ⬜ 샘플 부족 | — | 신호 3개 미만, 판단 불가 |

### 표시 지표
- 윈도우별 훈련/검증 승률 비교 바차트
- 수수료 반영 기댓값 (EV)
- 각 윈도우 상세 테이블

## 9. 리스크 분석

**[리스크 분석 실행]** 버튼으로 최근 1년 거래 기록 기반 리스크 지표를 계산합니다.

### 핵심 지표

| 지표 | 기준값 | 의미 |
|------|--------|------|
| **샤프 비율** | > 1.0 권장 | 수익 대비 전체 변동성 효율 |
| **소르티노 비율** | > 1.5 권장 | 수익 대비 하방 변동성 효율 |
| **최대 낙폭 (MDD)** | < 20% 권장 | 고점 대비 최대 자산 손실률 |
| **손익비 (Profit Factor)** | > 1.5 권장 | 총익 ÷ 총손 |
| **평균 수익 (거래당)** | — | 수수료 반영 평균 순수익 % |
| **평균 보유일** | — | 진입~청산 평균 일수 |

### 에쿼티 커브
100 기준 시작해서 각 거래 결과를 복리로 반영한 **가상 자산 성장 곡선**입니다.  
커브가 우상향할수록 안정적인 전략입니다.

### 수익률 분포 히스토그램
거래별 수익률 분포를 시각화합니다.  
정규분포에 가까울수록 예측 가능성이 높습니다.

## 10. 복리 수익 시뮬레이션

**[시뮬레이션 실행]** 버튼으로 전략 신호마다 복리 투자했을 때의 자산 성장을 시뮬레이션합니다.

### 입력값

| 항목 | 설명 |
|------|------|
| **초기 자본** | 시작 시 보유 자본 (KRW 또는 USD) |
| **포지션 크기** | 각 신호마다 현재 자본의 몇 %를 투자할지 (5~100%) |

### 켈리 공식
```
켈리 % = 승률(p) − 패율(q) ÷ 손익비(r)
```
- 이론적 장기 자산 성장을 극대화하는 최적 포지션 크기
- **하프 켈리(½ Kelly)** 사용이 실전 표준 — 낙폭이 절반으로 감소

### 시나리오 비교

| 시나리오 | 권장 대상 |
|----------|----------|
| **10% (보수)** | 리스크 회피형, 안정 우선 |
| **20% (중립)** | 일반 투자자 기본값 |
| **50% (공격)** | 고위험 감수, 극단적 변동성 |

### 출력 지표
- **최종 자본**: 시뮬레이션 종료 시 예상 자산
- **총 수익률**: `(최종 − 초기) ÷ 초기 × 100`
- **CAGR**: 연환산 복리 수익률
- **최대 낙폭**: 시뮬레이션 중 최대 자산 하락률 및 실금액
- **자본 성장 곡선**: 거래별 자산 변화 그래프
- **월별 수익률**: 월 단위 수익/손실 바차트

> ⚠️ 최대 낙폭 실금액이 감당 가능한 수준인지 반드시 확인하세요.

## 11. 5대 전략 상세

### 전략 1 — 장기이평선 + 라운드피겨 눌림목
- **조건**: 주봉 EMA(20) > EMA(60) > EMA(120) 정배열 + 일봉 종가가 SMA60 또는 SMA120 ±1.5% 이내
- **진입**: 가장 가까운 라운드 피겨(10의 배수) 지지선
- **적합 장면**: 강한 상승 추세 중 조정 매수 기회

### 전략 2 — 거래량 급감 + 도지/망치 캔들
- **조건**: 거래량 급증 돌파 이후 1~3일 조정 (거래량 < 돌파일의 30%) + 어제 도지 또는 망치형 캔들
- **진입**: 어제 종가 (도지/망치 확인 후 익일 시가 진입)
- **적합 장면**: 단기 눌림목에서 수요 재확인

### 전략 3 — 피보나치 0.5~0.618 황금 구간
- **조건**: 최근 30일 고점-저점 기준 피보나치 50~61.8% 되돌림 구간 내 종가
- **진입**: 현재 종가 (피보나치 황금 구간 진입 확인)
- **적합 장면**: 추세 반전 초입 또는 눌림목 지지 테스트

### 전략 4 — StochRSI 과매도 골든크로스
- **조건**: 주봉 SMA20 위 위치 + 일봉 StochRSI(14,14,3,3) K&D 모두 20 이하 + K가 D를 상향 교차
- **진입**: 골든크로스 발생 다음날 시가
- **적합 장면**: 과매도 구간에서 모멘텀 반전 신호

### 전략 5 — 박스권 돌파 후 S/R 리테스트
- **조건**: 50일 박스권 돌파(7일 이내, 거래량 1.5배 이상) + 어제 종가가 박스 상단(저항→지지 전환) ±2% 이내
- **진입**: 저항→지지 전환 레벨 (리테스트 확인 후)
- **적합 장면**: 박스 돌파 이후 첫 번째 되돌림 매수

## 12. 자주 묻는 질문

**Q. 스캔이 너무 오래 걸려요.**  
A. 최초 실행 시 거래소 전 종목 OHLCV를 수집하므로 1~3분이 소요됩니다. 이후 6시간 동안 서버 캐시가 유지되어 재실행 시 수초 이내로 단축됩니다.

**Q. 결과가 0개가 나왔어요.**  
A. 선택한 전략의 조건이 현재 시장 상황에 해당하는 종목이 없는 것입니다. 다른 전략이나 거래소를 시도해보세요.

**Q. 백테스트 승률과 워크포워드 승률이 크게 다른 이유는?**  
A. 백테스트는 전체 기간 집계라 과적합 위험이 있습니다. 워크포워드는 미래 데이터 없이 검증하므로 더 현실적인 지표입니다. **효율성 0.8 이상 (강건)** 종목을 우선하세요.

**Q. 켈리 공식 비율이 0%로 나와요.**  
A. 해당 전략의 과거 신호 중 손실 거래가 없거나, 거래 데이터가 부족한 경우입니다. 기본 10~20% 포지션으로 시뮬레이션하세요.

**Q. 이 시그널이 확정 매수 신호인가요?**  
A. 아닙니다. 이 도구는 **조건 탐색 + 통계적 분석**을 제공할 뿐이며, 실제 매수 결정은 반드시 본인의 판단과 리스크 관리 원칙에 따라야 합니다.
